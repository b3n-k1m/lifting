Using machine learning, this website discovers early symptoms of dyslexia by identifying whether a handwriting is written by a person or not, based on the common symptom of letter reversals, providing them doctor-free diagnosis. This web app can be used in large classrooms to decrease time spent by teachers on analyzing handwriting.
